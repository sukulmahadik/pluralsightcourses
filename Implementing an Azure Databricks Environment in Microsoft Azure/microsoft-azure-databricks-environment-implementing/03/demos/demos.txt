#Demo Code

This demo is based on the demo located on docs.microsoft.com at http://bit.ly/ETLdemo, Tutorial: Extract, transform, and load data by using Azure Databricks.

#Pre-Requisites
All of the pre-Requisites from the tutorial were implemented in a Azure subscription.

#Demo

To complete the demo in the tutorial, complete the following:
1. Create a new workspace and cluster in Azure Databricks
2. Import audience-notebook-clean.scala into your workspace
3. Modify the notebook w/ information from pre-requisite build out
    1. Notebook aligns to tutorial above so reference the tutorial if you have questions on the values you need to modify in the notebook.
4. Run through each cell of notebook.
5. To see what a functioning notebook looks like with results, import and view results-audience-data.scala
